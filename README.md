# Trabalho Prático de Estrutura de Dados
![C++](https://img.shields.io/badge/made%20with%20c++-%2300599C.svg?style=for-the-badge&logo=cplusplus&logoColor=white)

 1º trabalho prático de Estrutura de Dados feito no 3º período do curso de Sistemas de Informações (UFMG) em 2020.  
 Conteúdo abordado: TADs - Listas, Pilhas e Filas.
 
A ideia central do trabalho é explorar a utilidade dessas estruturas de dados em cenários em que o uso de cada uma é justificado em detrimento das demais e, portanto, conhecê-las a fundo é crucial. A fim de consolidar o conhecimento, ao invés de utilizar as implementações disponíveis nas bibliotecas da linguagem, os TADs foram implementados manualmente. A especificação completa do trabalho se encontra no arquivo 'Enunciado TP.pdf' e uma documentação do desenvolvimento do trabalho pode ser encontrada em 'documentação.pdf'.
